## 배포 (GitHub Pages)

push 또는 수동 실행 시 자동 배포됩니다.

**배포가 실패/취소될 때**  
- `The job was not acquired by Runner`  
- `Internal server error`  
- `Canceling since a higher priority waiting request for pages exists`  

위처럼 나오면 GitHub 쪽 일시적인 문제일 수 있습니다.  
몇 분 뒤 **Actions** 탭에서 해당 워크플로 → **Re-run all jobs** 한 번 더 시도해 보세요.
